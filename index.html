About

BEEODA is a suite of open-source software and educational materials for processing and analyzing earth observation data. It is based on a Linux Ubuntu Virtual Machine that contains a range of open-source image processing tools such as QGIS, Orfeo, GDAL, Python, R, etc. BEEODA is developed and maintained by Pontus Olofsson, Chris Holden and Eric Bullock who are all affiliated with the Department of Earth & Environment, Boston University. To explore BEEODA, please visit our GitHub repository.

Services

We offer to teach workshops using the BEEODA software and materials for analyzing earth observation data anywhere in the world. Workshop topics may include satellite image classification of land-cover, object-based classification of high-resolution imagery, detection and mapping of land-cover change in satellite imagery, time series analysis of satellite data, accuracy assessment of maps and area estimation of mapped land cover and change.

Near-Real Time Alerts

Our research group is in the process of a rigorous comparison of existing and operational near-real time alert systems with our own newly developed ones. Our two algorithms use daily MODIS data to alert of forest disturbances quickly and accurately.

Pre-Processing

Topics in the pre-processing portion of the workshops include:

  • Searching for and downloading data
  • FMask for cloud and shadow masking
  • LEDAPs for atmospheric correction
  • Image stacking
  • Creating image subsets
  • Image compositing
  • Introduction to Bash, Python, and QGIS

    • View the tutorial at: GitHub.

    • Date: July 2015

    Classification

    Topics to do with classification include:

  • Gathering training data appropriate for study
  • Using ancillary data to train machine learning algorithms
  • Creating maps with user-defined classes
  • Interactively tweeking training data and comparing classification algorithms to perfect map
  • Cleaning up map to remove isolated pixels

    • View the tutorial at: GitHub.

    • Date: July 2015

    Object-Based Analysis

    Image segmentation and object classification.

    Topics to do with segmentation include:

  • Segmenting images using spectral and textural properties
  • Direct classification of segments using spectral, geometric, and textural properties of segments
  • Pixel-based map classification using calculated segments for map filtering
  • Various multi-spectral segmentation techniques including Mean Shift and Watershed analysis

    • View the tutorial at: GitHub.

    • Date: July 2015

    Change Detection

    Data based methods.

    Topics to do with change detection include:

  • Image differencing
  • Principle Component Analysis
  • Multivariate Alteration Detection (MAD)
  • Direct classification of change

    • View the tutorial at: GitHub.

    • Date: July 2015

    Time Series Analysis

    Data exploration and change detection.

    Topics to do with time series analysis include:

  • Exploring spectral history of images at the pixel level
  • Calculating statistical models to fit to pixel trajectory
  • Using CCDC/YATSM to detect changes in temporal models
  • Classifying temporal segments for each pixel to track landcovers over time
  • Create change maps using time series algorithms
  • Using QGIS plugin specifically designed for time series analysis
  • Using image stacks for accuracy assessment purposes

    • View the tutorial at: GitHub.

    • Date: July 2015

    Error Estimation

    Accurate reporting of data.

    Topics to do with error estimation include:

  • How to appropriately draw a sample for asessign the accuracy of a map
  • How to intepret samples using Landsat images and Google Earth
  • How to calculate area-adjusted accuracy assessments using best practice methods found in literature

    • View the tutorial at: GitHub.

    • Date: July 2015
